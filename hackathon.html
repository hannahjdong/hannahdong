Rebranding MyAnimeList


When anime enthusiasts first enter MyAnimeList, they are greeted with tiny text and confusing layouts. One frequent user compares the site's web design to one from the early 2000s. As the team's designer, I wanted to give MyAnimeList a complete rebrand. Thinking about accessibility, I chose large images and a comfortable font size. The new home page centers an anime at the top and a row of recommendations based on the user's "swipes." When the user clicks on the anime, the anime "wiki" page appears. The page is filled with basic information, character details, and reviews. The user is also able to leave their own review. Other features of the rebrand are the profile page and the search page.
